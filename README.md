# 📚 504 Essential Words – IELTS Vocabulary App

A Progressive Web App for learning all 504 Absolutely Essential Words with English/German/Farsi translations, audio pronunciations, and weekly IELTS tests.

## 🚀 How to Launch

### Option 1: Windows
Double-click **`START-APP.bat`**

### Option 2: Mac / Linux
Double-click **`start-app.sh`** (or run `./start-app.sh` in terminal)

### Option 3: Any system with Python
```bash
python3 start-app.py
```

Your browser will open automatically at `http://localhost:8504`

### Option 4: Just open the HTML
Open **`app.html`** directly in Chrome (words work, but Install-as-App won't)

---

## 📲 Install as App (recommended!)

1. Launch with one of the methods above
2. Click the blue **"Install"** banner at the bottom
3. Or click the ⊕ install icon in Chrome's address bar
4. The app now appears in your Start Menu / Dock / Home Screen!
